# 진동모터 드라이버 회로 (MOSFET 2N7000)

**목적**: GPIO20의 약한 신호(3.3V, ~20mA)를 배터리 3.7V 강력한 신호로 증폭하여 진동모터 구동

---

## 부품 목록

| 부품 | 모델 | 수량 | 비고 |
|------|------|------|------|
| MOSFET | 2N7000 (TO-92) | 1 | N-channel, 손납땜 용이 |
| 다이오드 | 1N4148 | 1 | Flyback 보호, 역기전력 차단 |
| 저항 | 100Ω 1/4W | 1 | Gate 보호 저항 |
| 배터리 | 3.7V Li-Po 500mAh JST | 1 | 모터 전원 직결 |
| 모터 | Ø10mm 3V DC 진동모터 | 1 | 코인형, 무게 <2g |

---

## 회로도 (ASCII)

```
                          배터리 3.7V (+)
                               │
                               ├──────→ 모터 (+)
                               │           │
                               │       [1N4148]  ← Flyback 다이오드
                               │        (역방향)
                               │           │
            GPIO20 ─┬─[100Ω]──┬──→ Drain   모터 (-)
                    │         │
                    │    2N7000 MOSFET
                    │         │
                    └────┐ Gate
                        (TO-92)
                         │
                      Source ──→ GND (배터리 음극)
```

---

## 핀 정의 (2N7000 TO-92)

```
    1(G) - Gate
    │
2N7000 [정면 보기]
    │
    3(D) - Drain
    │
    2(S) - Source
```

| 핀 | 이름 | 연결 대상 | 목적 |
|----|----|---------|------|
| 1 | Gate (G) | GPIO20 (100Ω 거쳐서) | 신호 입력 |
| 2 | Source (S) | GND (배터리 음극) | 기준점 |
| 3 | Drain (D) | Motor (-) / 1N4148 애노드 | 출력 |

---

## 1N4148 다이오드 배치

```
1N4148 (DO-35, 작은 실린더)
│
├─ 띠 방향 = 캐소드 (-) ──→ Motor (+)
└─ 반대쪽 = 애노드 (+) ──→ Motor (-)

역방향 병렬 연결:
  Motor (-)  ───o─── Motor (+)
               1N4148 (역방향)
               띠 방향이 Motor (+)를 가리킴
```

---

## 동작 원리

### 상태 1: GPIO20 = HIGH (3.3V) → 모터 ON

```
GPIO20 = 3.3V
   ↓
Gate에 양전위 인가
   ↓
2N7000 도통 (ON) → Drain-Source 간 저항 ↓ (수십 Ω)
   ↓
배터리 전류 경로:
배터리(+) 3.7V 
   → Motor(+)
   → Motor(-)
   → 2N7000 Drain
   → 2N7000 Source (GND)
   ↓
모터 진동 시작 ⚙️
```

### 상태 2: GPIO20 = LOW (0V) → 모터 OFF

```
GPIO20 = 0V
   ↓
Gate에 0V 인가
   ↓
2N7000 차단 (OFF) → Drain-Source 간 저항 ↑ (매우 높음, MΩ)
   ↓
전류 차단
   ↓
모터 회전 스톱
   ↓
역기전력 발생 (모터 관성으로 인한 전압 스파이크)
   ↓
1N4148이 스파이크 제거 (역방향 도통, 에너지 소산)
   ↓
회로 보호 ✓
```

---

## 납땜 순서

1. **2N7000 핀 구부리기** (납땜 기판에 고정)
   - 1(G) 핀: 길게 유지
   - 2(S) 핀: GND로 연결
   - 3(D) 핀: Motor(-)로 연결

2. **100Ω 저항 납땜**
   - 한쪽 끝: GPIO20 점퍼선
   - 다른 쪽 끝: 2N7000 Gate(1) 핀

3. **1N4148 다이오드 납땜** (역방향 주의!)
   - 애노드(+, 띠 없는 쪽): Motor(-)
   - 캐소드(-, 띠 있는 쪽): Motor(+)

4. **배선 연결**
   - GPIO20 ──[100Ω]── 2N7000 Gate
   - 2N7000 Source ──── GND
   - 2N7000 Drain ──── Motor(-) 및 1N4148 애노드
   - Motor(+) ────────── 배터리 3.7V (+)
   - Motor(-) ────────── 위의 2N7000 Drain과 연결
   - 1N4148 캐소드 ─── Motor(+)

---

## 배선 체크리스트

```
[ ] 2N7000 Gate (1번 핀) ← GPIO20 (100Ω 직렬)
[ ] 2N7000 Source (2번 핀) → GND 연결
[ ] 2N7000 Drain (3번 핀) → Motor (-) 연결
[ ] 1N4148 애노드 (+, 띠 없음) → Motor (-)
[ ] 1N4148 캐소드 (-, 띠 있음) → Motor (+)
[ ] Motor(+) → 배터리 3.7V 직결
[ ] 모든 焊点 확인 (단락 없음)
```

## 펌웨어 코드 (MicroPython)

```python
from machine import Pin, PWM
import time

# GPIO20 PWM 설정
motor = PWM(Pin(20), freq=1000, duty=0)

# 모터 ON (100% 파워)
motor.duty(1023)  # 0~1023 범위
time.sleep(0.5)   # 0.5초 진동

# 모터 OFF
motor.duty(0)

# 진동 강도 조절 (PWM duty cycle)
for duty in range(0, 1024, 100):
    motor.duty(duty)
    time.sleep(0.2)
```

## 스펙

| 항목 | 값 | 비고 |
|------|-----|------|
| 입력 신호 | GPIO20, 3.3V | MicroPython PWM |
| 출력 전압 | 배터리 3.7V | MOSFET 드롭 < 0.2V |
| 최대 전류 | ~300mA @ 3.7V | 모터 정격 기준 |
| PWM 주파수 | 1~5kHz | 권장 1kHz |
| 온도 범위 | -20°C ~ +80°C | 2N7000 스펙 |

## 문제 해결

### 문제 1: 모터가 진동하지 않음

```
원인 분석:
1. GPIO20 출력 확인 (Thonny에서 on/off 테스트)
   → motor = PWM(Pin(20), freq=1000)
   → motor.duty(512)  # 50% 파워 테스트
   
2. 배터리 극성 확인 (멀티미터)
   → Motor(+) = +3.7V, Motor(-) = 0V?
   
3. 2N7000 방향 확인 (TO-92 1=G, 2=S, 3=D)
   → Gate가 GPIO20과 연결?
   → Source가 GND?
   
4. 회로 도통 확인
   → GPIO20 → 100Ω → Gate 도통?
   → Drain → Motor(-) 도통?
```

### 문제 2: 모터가 계속 진동 (OFF 안 됨)

```
원인:
1. Gate 누수 (100Ω 저항 불량)
   → 저항 교체
   
2. 2N7000 고장
   → MOSFET 교체
   
3. GPIO20이 HIGH 상태 고정
   → 펌웨어 확인
```

### 문제 3: 모터 진동이 약함

```
원인:
1. PWM duty 낮음
   → duty(1023)으로 100% 테스트
   
2. 배터리 전압 낮음
   → 배터리 충전 확인 (멀티미터)
   
3. 모터 불량
   → 배터리 직접 연결로 테스트
```
