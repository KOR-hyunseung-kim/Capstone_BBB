## 📌 MOSFET 드라이버 회로 (팀원 4 제작)

```
배터리 3.7V ─────────────── Motor (+)
                                │
                           [1N4148]
                                │
                         Motor (-) ── Drain
                                    MOSFET 2N7000
GPIO ──[100Ω]────────────── Gate
                               Source ── GND
```

**구성 방법**:
1. 만능기판 2×3cm에 2N7000 납땜
2. Gate에 100Ω 저항, GPIO 연결
3. Drain → Motor (-), Source → GND (배터리와 공통접지)
4. Motor (+) → 배터리 3.7V 직결 (LDO 라인과 분리)
5. 1N4148을 Motor 양단 역방향 병렬 연결 (역기전력 보호)
