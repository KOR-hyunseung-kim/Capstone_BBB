# BBB Flexible PCB - 납땜 부분만 정리 (진동모터 + 스위치 회로)

## 개요
- **납땜하는 부분**: 진동모터 회로 + 스위치 회로 (분압기)
- **점퍼선으로 연결**: ESP32-S3, MyoWare, MPU6050, 전원/GND 배포
- **목표**: 두 개의 독립적인 소회로를 Flexible PCB에 안정적으로 구성

## 2. 납땜 부분 1: 스위치 회로 (분압기)

### 회로도

```
     POWER RAIL (3.7V)
          ↓
       ┌──┴──┐
       │ [1kΩ] │ ← 납땜
       │      ├─→ 점퍼선 → ESP32-S3 A0 (ADC)
       │ [2kΩ] │ ← 납땜
       │      │
     GND RAIL
```

### 부품 목록

| 부품 | 값 | 개수 | 비고 |
|------|-----|------|------|
| 저항 1 | 1kΩ | 1개 | 1/4W |
| 저항 2 | 2kΩ | 1개 | 1/4W |
| 배터리 | 3.7V Li-Po | - | 점퍼선 연결 |

### 납땜 순서

#### **1단계: 배치 준비**

```
     Power Rail 위쪽
            ↓
      ┌─ [1kΩ] ─┬─ 점퍼선 연결점
      │          │
   Battery(+)    A0
      │          │
      └─ [2kΩ] ─┴─ GND
      
     GND Rail 아래쪽
```

**준비**:
- 1kΩ, 2kΩ 저항 2개를 준비
- 양쪽 선 끝을 벗겨냄 (약 5mm)
- Flexible PCB 위쪽과 아래쪽 위치 표시

#### **2단계: 1kΩ 저항 납땜**

```
위치: Power Rail 상의 분압기 시작점 (①번)

    Power(+) ──[연결]──[1kΩ]──[중점]
              (납땜)         (분압기 출력)
```

- 1kΩ 저항의 한쪽 끝을 Power Rail(배터리+)에 납땜
- 다른 쪽 끝을 자유 상태로 유지 (나머지 저항과 연결 예정)

**납땜 팁**:
```
인두 온도: 350°C
접촉 시간: 3초
용융 상태: 납이 광택나고 흘러내려야 함
```

#### **3단계: 2kΩ 저항 납땜**

```
위치: GND Rail 상의 분압기 끝점 (①번)

    GND(−) ──[연결]──[2kΩ]──[중점]
           (납땜)         (분압기 출력)
```

- 2kΩ 저항의 한쪽 끝을 GND Rail(배터리−)에 납땜
- 다른 쪽 끝을 자유 상태로 유지

#### **4단계: 분압기 중점 연결**

```
     [1kΩ]
      ↓
      ├─→ (점퍼선) → A0
      ↓
     [2kΩ]
```

- 1kΩ과 2kΩ의 **중점** (두 저항 사이)에 점퍼선 한 개 연결
- 이 점퍼선이 나중에 ESP32-S3의 A0 핀으로 연결됨

**연결 방식**:
```
방법 1: 두 저항을 한곳에 모으기
  1kΩ의 끝 ─┬─ [작은 납땜점]
  2kΩ의 끝 ─┘
               ↓ (점퍼선 연결)

방법 2: 각각 따로 점퍼선 연결 (권장)
  1kΩ의 끝 → [점퍼선1] → A0 입력 신호
  2kΩ의 끝 → [점퍼선2] → GND
```

## 3. 납땜 부분 2: 진동모터 회로 (MOSFET)

### 회로도

```
    POWER RAIL (3.7V)
         ↓
    진동모터(+)
         ↓
     MOSFET Drain
         │
    ──┬──┴──┬──
      │     │
    [100Ω] Gate ← 점퍼선 (D9로부터)
      │     │
    [1kΩ] Source
      │     │
    GND RAIL (0V)
         ↑
    진동모터(−)
```

### 부품 목록

| 부품 | 값/모델 | 개수 | 비고 |
|------|---------|------|------|
| MOSFET | AO3400 또는 2N7000 | 1개 | DIP-3 또는 SOT-23 |
| 저항 1 | 100Ω | 1개 | Gate 보호용 (스파이크 방지) |
| 저항 2 | 1kΩ | 1개 | Gate-Source 안정화 |
| 진동모터 | 3~5V | 1개 | 점퍼선 연결 |

### MOSFET 핀 정의

```
AO3400 (SOT-23) - 위에서 본 뷰:
  1 ─┐
     ├─ Gate (G)
  2 ─┤
     ├─ Source (S)
  3 ─┘
     └─ Drain (D)

또는 DIP-3:
  [1] Gate    [2] Drain   [3] Source
```

**확인**: 데이터시트 또는 부품 자체 라벨 확인 필수!

### 납땜 순서

#### **1단계: MOSFET 배치**

```
위치: GND Rail 우측 하단 (②번)

    MOSFET
    ├─ Gate (1번 핀)
    ├─ Drain (2번 핀)
    └─ Source (3번 핀)
```

#### **2단계: Drain 납땜 (Power 쪽)**

```
    Power Rail ──[Drain (2번)]──[진동모터(+)]
                    ↑
                  납땜점
```

- MOSFET Drain 핀을 Power Rail에 직접 납땜
- 또는 단선으로 연결해서 진동모터(+)로 배선

**선택**:
```
방법 A: Power → MOSFET Drain → 진동모터(+) (직렬)
        가장 간단, 권장

방법 B: Power → MOSFET Drain
        Power → 진동모터(+) (병렬)
        복잡하므로 비권장
```

#### **3단계: Source 납땜 (GND 쪽)**

```
    GND Rail ──[Source (3번)]──[진동모터(−)]
                   ↑
                 납땜점
```

- MOSFET Source 핀을 GND Rail에 납땜
- 진동모터(−)는 점퍼선으로 연결

#### **4단계: Gate 저항 연결 (아직 납땜 X)**

```
Gate (1번)
  ├─[100Ω]─┬─ (점퍼선) → D9
  │        │
  └─[1kΩ]──┘
           ↓
         GND Rail
```

- 100Ω 저항: Gate ← 점퍼선 (D9 PWM 신호)
- 1kΩ 저항: Gate ↔ Source 안정화

**수정 순서** (중요):
1. 100Ω 저항의 한쪽을 Gate(1번)에 납땜
2. 100Ω 저항의 다른 쪽을 자유 상태로 두기 (점퍼선 연결 예정)
3. 1kΩ 저항의 한쪽을 Gate(1번)에 납땜
4. 1kΩ 저항의 다른 쪽을 GND Rail에 납땜

## 4. 최종 배선 연결 (점퍼선)

### Power/GND 분배 점퍼선

```
배터리(+) 3.7V
    ↓
  [납땜 분압기 상단] ──○─ Power 배포점 1
  [납땜 MOSFET Drain] ──○─ Power 배포점 2
    └─→ 점퍼선 5개 → ESP32, MyoWare, MPU6050, LED 등

배터리(−) 0V
    ↓
  [납땜 분압기 하단] ──●─ GND 배포점 1
  [납땜 MOSFET Source] ──●─ GND 배포점 2
    └─→ 점퍼선 5개 → ESP32, MyoWare, MPU6050, LED 등
```

### 신호선 점퍼선

| 출발 | 연결 | 설명 |
|------|------|------|
| 분압기 중점 | ESP32-S3 A0 | ADC 입력 (스위치 감지) |
| ESP32-S3 D9 | MOSFET Gate | PWM 신호 (진동모터 제어) |
| ESP32-S3 D0 | MPU6050 SDA | I2C 신호 |
| ESP32-S3 D1 | MPU6050 SCL | I2C 신호 |
| 진동모터(+) | MOSFET Drain | 전원 (또는 분배점) |
| 진동모터(−) | MOSFET Source | GND |

## 5. 납땜 점검 체크리스트

### 분압기 회로 (스위치)

- [ ] 1kΩ 저항이 Power Rail에 납땜되었는가?
- [ ] 2kΩ 저항이 GND Rail에 납땜되었는가?
- [ ] 두 저항의 중점에 점퍼선이 연결 예정인가? (아직 인두 필요 없음)
- [ ] 멀티미터 저항 측정: Power ↔ GND = 약 667Ω (계산값: 1k || 2k)

### MOSFET 회로 (진동모터)

- [ ] Drain 핀이 Power Rail에 납땜되었는가?
- [ ] Source 핀이 GND Rail에 납땜되었는가?
- [ ] 100Ω 저항이 Gate(1번)에 납땜되었는가?
- [ ] 1kΩ 저항이 Gate와 GND 사이에 납땜되었는가?
- [ ] 멀티미터로 Gate 저항값 확인: 약 909Ω (100Ω || 1kΩ)

### 전체

- [ ] Power Rail 연속성 확인 (배터리+ ~ 분압기 ~ MOSFET Drain)
- [ ] GND Rail 연속성 확인 (배터리− ~ 분압기 ~ MOSFET Source)
- [ ] 모든 납땜점이 광택나고 균일한가?
- [ ] 배터리 극성 재확인 (+ 위, − 아래)

## 7. 전원 인가 후 확인

### 1단계: 무부하 테스트

```
배터리만 연결 (MCU 없음):
□ Power/GND 전압: 3.7V ± 0.1V (멀티미터 측정)
□ 지속 시간: 30초 관찰 (이상 현상 없는가?)
```

### 2단계: MCU 추가

```
ESP32-S3 연결:
□ 전류: ~40mA (idle)
□ 부팅 로그: USB 시리얼 포트에서 확인
□ 온도: 미온한 정도 (뜨겁지 않은가?)
```

### 3단계: 센서 추가

```
MyoWare + MPU6050 연결:
□ I2C 스캔 성공: MPU6050 주소 0x68 감지
□ ADC 읽음: A0 핀에서 분압기 신호 수신
```

### 4단계: 진동모터 테스트

```
MOSFET + 진동모터 연결:
□ GPIO D9에 PWM 신호 인가 (예: 50% duty)
□ 진동모터 작동 확인 (울림 또는 진동)
□ PWM 주기: 1kHz 권장 (너무 낮으면 끊김)
```

## 8. 문제 해결

| 문제 | 원인 | 해결책 |
|------|------|--------|
| 배터리 빨리 방전 | Power/GND 단락 | 멀티미터로 저항값 확인 (무한대 아님 = 단락) |
| I2C 미감지 | SDA/SCL 선 바뀜 또는 단락 | 점퍼선 위치 재확인, 연속성 테스트 |
| ADC 값 이상 | 분압기 배선 오류 | Power/GND 연결점 재확인 |
| 진동모터 미작동 | MOSFET 방향 반대 또는 Gate 미연결 | Gate 점퍼선 재확인, MOSFET 데이터시트 재확인 |
