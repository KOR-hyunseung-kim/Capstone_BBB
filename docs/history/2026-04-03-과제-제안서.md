## I. 과제 개요

### 1. 과제 추진 배경

#### 1.1 산업 안전 문제
- **근골격계 질환 증가**: 반복적인 동작과 장시간 작업으로 인한 근피로가 누적되어 근골격계 질환 발생
- **기존 장구의 한계**: 현재 산업용 안전 장구는 외부 충격 보호에만 집중되어 있으며, 근피로도의 **정량적 모니터링이 불가능**
- **작업자 안전 관리**: 피로도에 기반한 사전 예방 및 휴식 지도의 필요성 대두

#### 1.2 오염 환경에서의 인터페이스 문제
- **현장 작업 특성**: 기름, 화학약품, 먼지 등이 손에 묻은 상태에서 노트북/태블릿 조작
- **기기 오염 및 고장**: 직접 터치로 인한 장비 손상 및 위생 문제
- **핸즈프리 솔루션 필요**: 신체 동작만으로 기기를 제어할 수 있는 비접촉식 입력 기술의 현장 도입 필요

#### 1.3 기술적 기회
- **웨어러블 센서 성숙화**: EMG(근전도), IMU(관성) 센서의 소형화·저가화
- **무선 통신 표준화**: Bluetooth HID 프로필이 주요 OS(Windows/Mac/Linux)에서 기본 지원
- **임베디드 성능 향상**: ESP32-S3 같은 저전력 MCU에서 신호처리 및 무선 통신 동시 구현 가능

### 2. 개발 목표

#### 2.1 기능 목표
1. **Safety Mode**: 실시간 근피로도 모니터링
   - EMG 신호로부터 피로도 지수(0~100%) 추출
   - 80% 임계치: 경고(노란색 LED, 약한 진동)
   - 95% 임계치: 긴급(빨간색 LED, 강한 진동)

2. **Control Mode**: 비접촉식 HID 제어
   - IMU(팔 기울기) → 커서 이동 (pitch/roll 각도)
   - EMG Spike(주먹 쥐기) → 마우스 클릭 (BLE HID 이벤트)
   - Windows/Mac/Linux 표준 마우스로 인식

#### 2.2 성능 목표
| 항목 | 목표값 |
|------|--------|
| **지연시간** | 50ms 이내 (센서 입력 → 화면 반응) |
| **센서 샘플율** | EMG 1kHz, IMU 100Hz |
| **배터리 지속시간** | 2시간 이상 연속 동작 |
| **무게** | 100g 미만 |
| **오검출율** | <5% (EMG Spike 오인식) |
| **커서 드리프트** | <2px/s (정지 상태에서) |

#### 2.3 기술 목표
- Bluetooth HID 프로필 표준 구현
- 20~500Hz Band-pass 필터링 (EMG 노이즈 제거)
- FFT 기반 주파수 특성 분석 (Median Frequency 추출)
- Complementary Filter를 이용한 IMU 센서 융합

### 3. 개발 내용

#### 3.1 하드웨어 (H/W)
| 부품 | 모델 | 용도 |
|------|------|------|
| **MCU** | Seeed XIAO ESP32-S3 | 센서 제어, 통신, WiFi/BLE |
| **EMG 센서** | MyoWare 2.0 | 근전도 신호 수집 |
| **IMU 센서** | MPU6050 (6-Axis) | 팔 기울기 감지 |
| **배터리** | 3.7V Li-Po 300mAh+ | 무선 전원 |
| **피드백** | 코인 진동모터 | 촉각 경고 알람 |
| **디스플레이** | 0.96" OLED (선택) | 상태 표시 (선택사항) |

**설계 제약**:
- 총 무게 100g 미만 (경량화 최우선)
- BLE HID 또는 WiFi만 사용 (USB 유선 금지)
- 2시간 이상 연속 동작 (배터리 관리 필수)
- 3D 프린팅 하우징(FDM PLA/PETG) 기준 설계

#### 3.2 펌웨어 (F/W) — ESP32-S3 MicroPython
```
firmware/
├── sensor/
│   ├── emg.py       # MyoWare 2.0 ADC 드라이버
│   └── imu.py       # MPU6050 I2C 드라이버
├── algo/
│   ├── filter.py    # Band-pass, Complementary 필터
│   └── dsp.py       # FFT, Median Frequency
├── comm/
│   ├── wifi.py      # WiFi UDP 송신 (센서 데이터)
│   └── ble.py       # BLE HID 수신 (제어 명령)
├── ui/
│   ├── motor.py     # GPIO PWM 진동모터 제어
│   ├── led.py       # RGB LED (선택)
│   └── oled.py      # I2C OLED (선택)
└── main.py          # 모드 전환, 메인 루프
```

**주요 기능**:
- WiFi UDP로 센서 raw 데이터 송신 (PC 측 DSP 처리용)
- BLE HID 명령 수신 (PC → 외부 기기 제어 중계)
- 모드 전환 (Safety ↔ Control) 버튼 인터럽트 처리
- 저전력 설계 (Light Sleep 적극 활용)

#### 3.3 PC 도구 (Tools) — Python 3.11+
```
tools/
├── receiver.py      # WiFi UDP 서버 (센서 데이터 수신)
├── algo/
│   ├── filter.py    # 신호 처리 (필터링, FFT)
│   ├── fatigue.py   # 피로도 판정 로직 (80%/95% 임계치)
│   └── cursor.py    # IMU → 커서 좌표 변환
├── hid_controller.py # BLE HID 명령 발신 (pyautogui)
└── main.py          # 통합 서버 메인 루프
```

**주요 기능**:
- WiFi UDP로부터 EMG/IMU raw 데이터 수신
- 실시간 신호 처리 (필터링, FFT, 피로도 계산)
- pyautogui를 이용한 커서 제어
- 피로도 임계치 도달 시 BLE HID 진동/LED 제어 명령 송신

## III. 기대 효과

### 1. 기대 효과

#### 1.1 산업 안전 분야
- **근피로도 정량화**: 기존에 불가능했던 근피로도를 **정량적 지수(%)로 측정**
  - 작업자 건강 관리의 과학적 근거 제공
  - 근골격계 질환 예방 프로그램 데이터 기반 수립 가능

- **사전 예방 체계 구축**: 
  - 80% 임계치 경고 → 조기 휴식 개입
  - 95% 긴급알람 → 무리한 작업 즉각 중단
  - 결과: 산업재해 감소 및 작업 효율성 증대

#### 1.2 HCI (Human-Computer Interaction)
- **오염 환경 친화적 인터페이스**:
  - 손이 더러운 상태에서도 노트북/태블릿 제어 가능
  - 기기 오염 및 고장 감소 → 유지비 절감
  - 위생 개선 (비접촉식)

- **신체 동작 기반 입력**:
  - 팔 기울기(IMU) → 커서 이동
  - 주먹 쥐기(EMG Spike) → 클릭
  - 새로운 사용자 경험(UX) 가능성 열림

#### 1.3 기술적 성과
- **웨어러블 신호처리 기술**:
  - 실시간 EMG/IMU 센서 융합
  - Band-pass 필터링 + FFT 기반 특징 추출
  - 저지연(50ms), 저전력(2시간+) 임베디드 구현

- **표준 기반 무선 통신**:
  - Bluetooth HID 프로필 표준 구현
  - 플랫폼 독립적 (Windows/Mac/Linux 모두 호환)
  - 확장성 높은 아키텍처 (WiFi UDP + BLE 분리)

#### 1.4 경제적 효과
- **비용 절감**:
  - 저가 부품 기반 ($150 이하 BOM)
  - 대량생산 시 $50~80 목표
  
- **시장 기회**:
  - 산업용 웨어러블 시장 (건설, 물류, 제조)

   - 다중 채널 EMG (양팔 모니터링)
   - AI 기반 개인 맞춤 피로도 모델 (딥러닝)
   - 클라우드 데이터 분석 (그룹 피로도 통계)

#### 2.3 장기 (2027년 이후)
1. **생태계 확대**
   - 다양한 산업 현장 맞춤형 솔루션 (의료, 스포츠, 재활)
   - 오픈소스 공개 (GitHub) → 커뮤니티 기여 유도
   - 제3자 개발자를 위한 API/SDK 제공

2. **학문적 기여**
   - 웨어러블 센서 신호처리 교육 자료 개발
   - 오픈데이터셋 공개 (EMG/IMU 샘플 데이터)
   - 후속 연구 기초 제공 (근피로도 예측 모델, 개인화 칼리브레이션 등)

3. **사회적 영향**
   - 산업재해 감소 → 노동자 건강 보호
   - 비정규직/계약직 작업자 건강 관리 기준 제시
   - 산업 안전 문화 개선

#### 2.4 구체적 활용 사례

**Case 1: 물류 창고**
- 하역 작업자 5명 → 5개 아대 배포
- 일일 피로도 데이터 수집 & 분석
- 휴식 일정 과학적 수립 → 작업 효율성 20% 향상

**Case 2: 건설 현장**
- 콘크리트 타설 작업 모니터링
- 과로 예방 → 산재 감소
- 작업 일정 재편성 데이터 제공

**Case 3: 재활치료**
- 뇌졸중 환자 상지 복구 훈련
- EMG 신호로 근력 회복도 정량화
- 훈련 프로그램 개인화 피드백

**Case 4: 게이밍/VR**
- 손 제스처 인식 (주먹 쥐기 = 클릭)
- 새로운 게임 컨트롤러 경험
- 신체 활동 기반 엔터테인먼트
