## 완료된 작업

### 1. EMG 센서 (MyoWare 2.0)
- **상태**: ✅ 테스트 완료
- **결과**: ENV 포트 사용 → 신호 범위 정상 확인
  - 정지 상태: 150~350 ADC
  - 강한 수축: 2000~3500 ADC
- **GPIO**: GPIO1 (ADC1_CH0)
- **샘플링**: 1kHz (또는 50Hz ENV 신호 사용)

### 2. LED 모듈
- **상태**: ✅ 테스트 완료
- **결과**: 단색 LED + RGB LED 정상 동작
  - 단색 LED: GPIO18 + 220Ω 저항
  - RGB LED: GPIO16(R), GPIO17(G), GPIO20(B) PWM 제어
- **기능**: 모드 표시, 피로도 레벨 시각화 가능

### 3. 택트 스위치 회로
- **상태**: ✅ 회로 구성 완료, GPIO 입력 테스트 계획
- **설정**: GPIO21 입력
- **분압 저항**: 100Ω × 2 사용 가능 (또는 4.7kΩ + 10kΩ 권장)
  - 3.7V → 1.85V (또는 2.5V) 변환
  - 전류 소비: 18.5mA (100Ω) 또는 0.2mA (권장 저항)
- **테스트**: Thonny REPL에서 GPIO21 값 모니터링 확인 대기

## 결정 사항

### IMU 센서 교체 결정 ❌ → ✅ 신규 구매

**GY-521 (MPU6050) 문제점**:
- 신호가 매우 불안정 (값 변동 심함)
- I2C 노이즈로 인한 데이터 불일치
- 저가형 모듈의 품질 이슈

**교체 대상** (추천 순서):
1. **ICM-20602** (향상된 MPU6050 호환, 더 안정적)
2. **LSM6DSOX** (ST 센서, 최고 품질)
3. **재평가**: AD0, INT 핀 활용 등의 대안 시도
