Problem Statement

왜 BBB가 필요한가?

기존 안전 솔루션이 해결하지 못한 3가지 문제

⚠️

근피로 감지 수단 없음

근골격계 질환은 국내 직업병의 약 70%를 차지. 물류·제조 현장에서 근피로가 누적되면 물건을 떨어트리거나 부상으로 이어지지만, 이를 실시간 경보하는 현장용 웨어러블이 없음.

🧤

현장 오염 · 양손 사용 문제

기름·화학약품이 묻은 장갑을 착용한 채 공구를 잡고 있는 상황에서 노트북·태블릿을 조작하면 정밀 기기 오염·고장. 공구를 내려놓지 않고 조작할 수단이 필요.

🤚

비접촉식 입력장치 부재

음성인식은 소음 환경에서 오인식, 풋스위치는 이동 중 불가, Eye Tracker는 고가(50만원↑). 산업 현장에서 실용적으로 쓸 수 있는 신체 동작 기반 입력장치가 없음.

Demo Scenario A

Safety Mode

근골격계 질환 예방

물류 창고에서 하루 500회 이상 반복 하역 작업을 할 때 — 착용 후 아무런 조작 없이 자동 감지

1

작업 시작 — BBB 착용

아대 착용 후 전원 ON → WiFi로 노트북 모니터링 스크립트에 자동 연결 · 기준 MF 측정 시작

2

피로도 누적 — 자각 없이 진행 중

반복 들기 1.5시간 경과 → EMG Median Frequency 저주파 이동 → 본인은 아직 피로를 자각 못함

3

1단계 경보 — 80% 도달

손목에 진동 1회 + LED 노란색 → "지금 5분 쉬세요" 신호. 자연스럽게 휴식 결정

95% 도달 시 강진동
4

결과 — 선제적 휴식으로 부상 예방

오전 중 2~3회 자발적 휴식 → 임계치 초과 없음 → 전완 건염 발생 위험 대폭 감소. 미착용 시 동일 작업 1개월 반복 후 건염 진단 사례와 비교

Demo Scenario B

Control Mode

오염 환경 핸즈프리 제어

공구를 잡은 채 매뉴얼(AMM)을 수십 회 확인해야 하는 정비 작업을 할 때 — 공구를 내려놓지 않고 조작

1

상황 — 양손이 묶인 정비 현장

장갑 착용 + 양손에 공구. 매뉴얼 다음 페이지 확인 필요 → 기존에는 공구 내려놓기 → 터치패드 조작 → 다시 집기 (~15초 낭비 / 1회)

2

커서 이동 (IMU 팔 기울기)

팔을 원하는 방향으로 기울이면 MPU6050 pitch/roll → WiFi → pyautogui.move() → 커서 이동. 공구 그대로 유지

3

클릭 (EMG 주먹 쥐기)

주먹을 순간 꽉 쥐면 EMG 스파이크 감지 → WiFi → pyautogui.click() → 다음 페이지. 전체 소요 시간 ~2초

4

결과 — 작업 흐름 유지

기기를 한 번도 건드리지 않고 정보 확인 완료. 하루 50회 조회 시 약 10분 절약 · 오염 전파 Zero · 정밀 기기 고장 방지
