## 완료된 작업

### 기획 & 아키텍처 (2026-04-02)
- BBB 개요: Safety Mode (근피로 경보) + Control Mode (핸즈프리 커서 제어)
- 아키텍처 확정: 노트북 Python이 모든 연산 담당, ESP32-S3는 Sensor Hub + IO Controller
- 통신: WiFi UDP (LAN ~1~5ms 레이턴시), BLE HID 미사용
- 커서 제어: 노트북 `pyautogui` 직접 호출

### H/W BOM (2026-04-02)
- 핵심 부품 선정 완료 → `docs/bom.md`
- 총 예산: ~₩232,000 (예비비 포함)
- **선행 주문 필요**: MyoWare 2.0 × 2 (SparkFun DEV-21267, 해외 1~2주 배송)
- MOSFET 회로 결정: IRLML6244 + 1N4148 + 100Ω (진동모터 드라이버)

### 기획 보강 (2026-04-02)
- 타겟 페르소나 × 2 정의 (물류 작업자, 항공정비 작업자)
- 국내 근골격계 질환 현황 데이터 정리
- 기존 해결책 한계 분석 (피트니스 밴드, 임상 EMG, 음성인식 등)
- Safety/Control 모드 통합 근거 문서화
- 세부 시나리오 타임라인 작성 → `docs/specs/project-overview-v2.md`
